# Changelog

本文件记录用户可见的功能变化。技术细节和代码修改请参见 git log。

---

## v0.12.0 — 2026-06-24

### 摄影集章节
- 摄影集内新增 **章节标签**——在照片之间插入章节标题，组织照片叙事结构
- 章节以标签形式展示在照片网格上方，可随时添加和删除
- 导出 PDF 和 Markdown 时章节标题自动出现在照片之间

---

## v0.11.0 — 2026-06-23

### 摄影集管理
- **编辑摄影集名称**——点击标题旁的铅笔图标直接修改，Enter 保存，Esc 取消
- **批量移入摄影集**——在多选模式下勾选照片，一键加入已有摄影集
- **批量移出摄影集**——在摄影集内多选照片后移出，照片回到首页
- **删除摄影集**——删除摄影集同时将所有照片移回首页，不删除照片文件

### Markdown 导出
- 新增 **Markdown 格式导出**——摄影集导出为可编辑的源文件（zip 压缩包）
- 导出内容：book.md（Markdown 文本）+ images/ 目录（原始图片）
- 与 PDF 导出的区别：PDF 是最终出版物（含版式），Markdown 是可编辑源文件（仅内容）

### 文档
- 新增 **用户说明书**（USER_GUIDE.md）——面向摄影师的完整使用指南
- 新增 **更新日志**（CHANGELOG.md）——用户可见的功能变化记录

---

## v0.10.0 — 2026-06-16

### 首页布局改进
- 首页重构为双栏布局：左侧导入区 + 摄影摘要，右侧数据概览
- 新增自然语言摄影摘要——告诉你的拍摄节奏和器材偏好，无图表、无百分比
- 新增精简数据概览——总数、近 30 天、拍摄热力图、常用设备名称
- 统计分析移至独立页面（导航栏 → 统计分析），首页不再是数据面板

### 筛选系统升级
- 新增 **时间筛选**——8 种预设：全部、今天、7 天、30 天、90 天、本年、指定年份、自定义日期范围
- 新增 **焦段筛选**——6 类摄影语言：超广角 / 广角 / 标准焦段 / 中长焦 / 长焦 / 超长焦
- 新增 **备注筛选**——筛出有备注或无备注的照片
- 新增 **关键词搜索**——搜索文件名和备注内容
- 新增 **4 种排序**——拍摄时间（新→旧 / 旧→新）、导入时间（新→旧 / 旧→新）

## v0.9.0 — 2026-06-14

### 摄影语言分析
- 新增 **等效焦段引擎**——自动将不同画幅的焦段换算为 35mm 等效值
- 新增 **摄影语言分布**——6 类焦段柱状图，直观展示你的焦段偏好
- 新增 **摄影语言演变**——12 个月时间轴，追踪焦段偏好如何随时间变化
- 新增 **摄影习惯洞察**——自然语言总结你的摄影习惯，如"你主要使用广角拍摄"
- 统计分析面板升级为五段式布局（活跃度 / 相机 / 镜头 / 摄影语言 / 演变趋势）
- 新增活动计数卡片：30 天 / 90 天 / 今年累计 / 全部照片数

---

## v0.8.0 — 2026-06-12

### 批量操作
- 新增 **文件夹导入**——一次选择整个文件夹批量导入照片
- 新增 **批量上传**——一次可选多张照片，自动分批处理，显示进度条
- 新增 **多选模式**——在瀑布流中勾选多张照片，批量删除或移出摄影集
- 新增 **全选 / 反选**——快速选择或取消所有照片

### 存储模式
- 新增 **引用模式**（默认）——导入时仅记录文件路径，不复制文件
- 删除操作根据存储模式区分提示：引用模式仅删记录，复制模式删文件
- 摄影集内删除照片改为"移出摄影集"，照片回到首页，不删除文件

### 摄影分析
- 新增 **摄影习惯洞察**卡片——规则引擎生成，非 AI，完全离线
- 新增 **时段分布**——上午 / 下午 / 傍晚 / 夜晚四个时段的拍摄统计

## v0.7.0 — 2026-05-31

### 摄影书导出
- 新增 **摄影集 PDF 导出**——将已发布的摄影集导出为排版精美的摄影书
- PDF 支持 4 种画面比例：4:5 / 1:1 / 3:2 / 2:3
- 导出内容包含：封面标题页、照片页（居中 + 留白）、备注说明、页码
- 中文宋体排版，极简黑白灰设计

### 其他改进
- 新增 **每日拍摄热力图**——GitHub 贡献图风格，一眼看清拍摄节奏
- 修复系列 PDF 排版问题，中文显示、图片比例、页面溢出等

- 新增 **排序**——按拍摄时间筛选（新→旧 / 旧→新）
- 照片备注预览显示在卡片底部
